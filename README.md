🌱 AI Plant Disease Detection

An AI-powered web application built with Streamlit that helps identify plant diseases from leaf images and provides treatment suggestions along with weather information.

Features

- 🌿 Upload a plant leaf image
- 📷 Capture a photo using your camera
- 🤖 AI-based disease prediction
- 💊 Disease treatment and care recommendations
- 🌦️ Weather information
- 🎨 Simple and user-friendly interface

Technologies Used

- Python
- Streamlit
- Pillow (PIL)
- Requests
- Machine Learning

Installation

1. Clone the repository:
   
   git clone https://github.com/your-username/your-repository.git

2. Navigate to the project folder:
   
   cd your-repository

3. Install the required packages:
   
   pip install -r requirements.txt

4. Run the application:
   
   streamlit run app.py
